# Backend 

## 1. Create virtual evironment 

**For windows and mac**
   
   ```cmd
   python -m venv venv
   ```
   
## 2. Run virtual environment 

**Windows** 

```cmd
venv/Scripts/Activate
```

**Mac** 

```zsh
source venv/bin/activate
```

## 3. install requirements 

```
pip install -r requirements.txt
```

## 4. Runserver

```src 
python manage.py runserver
```
